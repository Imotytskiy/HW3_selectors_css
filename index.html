СЕЛЕКТОРИ CSS

1. Універсальний селектор спрацьовує для всіх елементів html. css - * {опис правил}, html - всі елементи на сторінці, для яких нема інших правил css.
2. Будь-який елемент html може бути селектором. Приклад <p>, <span>,<div>,<h1> тощо. css - p { опис правил}, html елемент <p>, якщо більше елементів <p> на сторінці до всіх них.
3. Селектор ідентифікатор має вищий преорітет над класом (призначений для стилізації одного елемента) - це унікальне ім'я елементу зустрічається в коді один раз. css - #ім'я {правила стилю }. html - <елемент id="ім'я">.
4. Класс використовується,коли небхідно декілька різних елементів стилізувати однаково. css - .назва класу {опис стилю}, html - визначається через атрибут (тег class="назва класу").
5. До елементів можна використовувати декілька класів,якщо є співпадіння значеннь в класі застосовується з останього класу (будь-яку кількість класів можна підключити). Можна описати різні стилі в різних класах і разом підключати до елементів. Спрощує стилізацію. html (тег class="назва класу назва класу").
6. Вкладений селектор, стиль застосувується для вкладеного елементу. css - елемент1 елемент2 {опис стилю;}, html - елемент1 застосування стилю до вкладеного елементу2 в елемент1.
7. Дочірний селектор, стиль застосовується тільки до дочірнього елементу. css - елемент1 > елемент2 {опис стилю;}, html - елемент1 застосування стилю до дочірнього елементу2,так само вкладений (батківський елемент) в елемент1.
8. Cусідні селектори, стиль застосовується до сусіда. css - елемент1 + елемент2 {опис стилю;}, html - елемент1 застосування стилю до сусіда елементу2.
9. Споріднені селектори, стиль застосовується як сусідні селектори + якщо є такі самі елементи першого рівня(не батьківські, не вкладені) до них також. css - елемент1~елемент2 {опис стилю;}, html - елемент1 застосування стилю до сусіда елементу2 та до елементів2, якщо вони є на цьому рівні ще сусіди.
10. Селектори атрибути [attr], якщо в елементі є атрибут до нього можна прив'язатися. css - [<атрибут>] { Опис правил стилю }, html - елемент[<атрибут>]. Всі елементи з таким атрибутом на сторінці отримають відповідний стиль.
11. Селектори атрибути [attr='value'], якщо значення починається з зазначеного тексту. css - [атрибут^="http:"] { Опис правил стилю }, html - елемент[<атрибут>="http://www.q.er"]
12. Селектори атрибути [attr$='value'], якщо значення закінчується текстом. css - [атрибут$=".er"] { Опис правил стилю }, html - елемент[<атрибут>="http://www.q.er"]
13. Селектори атрибути [attr*='value'], якщо значення будь-яка чатина тексту. css - [атрибут^=".q"] { Опис правил стилю }, html - елемент[<атрибут>="http://www.q.er"]
14. Селектори атрибути [attr~='value'], якщо значення має пробіл після назви. css - [атрибут~="big"] { Опис правил стилю }, html - елемент[<атрибут>="big star"]
15. Селектори атрибути [attr|='value'], якщо значення має дефіс. css - [атрибут~="big"] { Опис правил стилю }, html - елемент[<атрибут>="big-star"]
